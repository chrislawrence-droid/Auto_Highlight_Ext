Auto-Highlight Extension Demo

How to Test Auto-Highlight

  1. Install the extension in Edge/Chrome
  2. Click the extension icon
  3. Enable "Auto-Highlight Mode"
  4. Add some default terms like "highlight", "extension", "demo"
  5. Close the interface and refresh this page
  6. Your terms should be automatically highlighted!

Sample Text for Testing

This is a demonstration page for the auto-highlight extension. The extension can automatically highlight important terms like highlight, extension, and demo without requiring manual clicks each time.

When you enable auto-highlight mode and set default terms, the extension will automatically scan every page you visit and highlight those terms with unique colors. This is especially useful for:

  • Medical professionals reviewing patient records
  • Researchers analyzing documents
  • Students studying specific topics
  • Anyone who frequently searches for the same terms

Example Highlighted Terms

Here are some terms that might be useful to auto-highlight:

  • patient - for medical records
  • diagnosis - for clinical notes
  • treatment - for care plans
  • medication - for drug lists
  • symptoms - for patient complaints

The extension works by injecting a content script into web pages that searches for text nodes containing your specified terms. It then wraps matching text in colored highlight spans while preserving the original page structure.

Technical Features

This extension includes several advanced features:

  • Multi-term highlighting - Search for multiple terms simultaneously
  • Unique color coding - Each term gets a distinct color
  • Case-insensitive search - Find terms regardless of capitalization
  • Privacy-focused - No external API calls, everything runs locally
  • Auto-highlight mode - Automatically highlight on every page
  • Persistent settings - Your preferences are saved between sessions

To get the most out of the auto-highlight feature, consider setting up terms that are relevant to your daily work. For example, if you're a healthcare provider, you might want to auto-highlight terms like "vital signs", "lab results", "medication", and "allergies".

Testing Instructions

Try these steps to test the auto-highlight functionality:

  1. Set default terms: "highlight", "extension", "demo", "terms"
  2. Enable auto-highlight mode
  3. Navigate to a different page or refresh this one
  4. Watch as your terms are automatically highlighted!

Remember, the extension only works on regular web pages, not on browser settings pages or special URLs. If you're having trouble, check the browser console for any error messages.
